Wie ben ik?

Hi! Ik ben Sarah, 19 jaar en eerstejaars student Communication and Multimedia Design (CMD) aan de Hogeschool van Amsterdam. Tijdens mijn studie heb ik al veel kennisgemaakt met verschillende vakgebieden zoals visual design, web development en user research. Ik ontdek steeds meer waar mijn talenten liggen! Ik vind het vooral leuk om te werken aan de design en technische kant. Ik krijg energie van projecten waarin ik mijn creativiteit en technische skills kan combineren. In de toekomst zou ik mij graag verder willen verdiepen in front-end development. In mijn vrije tijd vind ik het leuk om nieuwe dingen uit te proberen of skills beter onder de knie te krijgen. Ook speel ik graag spelletjes met familie en vrienden, vind ik het leuk om te reizen, te koken, naar muziek luisteren en om te lezen.

Dakine-HCI

Tijdens dit vak (Human Computer Interactions) heb ik veel geleerd over de theorie van User Interfaces. Ik heb veel van de theorie opgeslagen en probeer dat nog steeds bij andere projecten en vakken goed toe te passen.

Voor het merk Dakine (de ‘opdrachtgever’) heb ik een interactieve interface moeten maken. Ik heb nagedacht over een juiste flow, foutmeldingen en de layout. Ik heb hiervoor voor het eerst in Figma gewerkt en daardoor ook veel tijd gestopt in het leren van het programma.

Podcast-Internetstandaarden

Voor het vak Internetstandaarden ben ik voor het eerst aan de slag gegaan met HTML en CSS. Ik leerde hoe je een website goed opbouwt, netjes styled en zorgt dat alles werkt zoals het hoort. In het begin was het even zoeken hoe alles precies samenhangt en werkt, maar uiteindelijk vond ik het super leuk om te zien hoe mijn ideeën echt zichtbaar werden in de vorm van een website. Ik heb tijdens dit vak veel kunnen leren. Voor de eindopdracht heb ik mijn eerste volledige website gemaakt.

Pizza-Inleiding Programmeren

In het vak Inleiding Programmeren ben ik voor het eerst aan de slag gegaan met JavaScript. Ik leerde hoe je een website interactiever kunt maken met dingen zoals variabelen, functies, if-statements en loops. In het begin vond ik het soms best lastig, maar na veel proberen en oefenen begon ik het steeds beter te begrijpen en werd het zelfs erg leuk om ermee te experimenteren. Voor de eindopdracht heb ik een interactieve website gemaakt waarin ik JavaScript gebruik om pizza beleg toe te voegen aan de pizza.

Informatie Architectuur

Tijdens het vak Informatie Architectuur heb ik geleerd hoe belangrijk een goede structuur en logische opbouw is voor interfaces.
